## 🎯 **HOW IT WORKS**

### **Scenario: BUY → SELL Bias Flip**

**Initial State (BUY bias):**
```json
{
  "live": [
    {
      "strategy": "breakout_4_tp10.0_sl50.0",
      "product": "GBPEUR_C",
      "buy": true,
      "sell": false,
      "activated_for_bias": "BUY",
      "activated_at": "2026-02-09T17:00:00"
    }
  ]
}
```

**Bias Flip Detected (BUY → SELL):**
1. ✅ `automated_strategy_picker.py` detects bias change
2. ✅ Updates `_targeted_strategies.json` with SELL bias
3. ✅ Bias monitor detects file change within 5 seconds
4. ✅ Modal popup appears (if `ai_picker_prompt: true`)

**User Clicks "Activate Strategy":**
1. ✅ Frontend calls `/api/deactivate_wrong_bias` with `current_bias: SELL`
2. ✅ Backend finds GBPEUR_C strategy (has `activated_for_bias: BUY`)
3. ✅ Backend deactivates it:
   ```json
   {
     "strategy": "breakout_4_tp10.0_sl50.0",
     "product": "GBPEUR_C",
     "buy": false,
     "sell": false,
     "activated_for_bias": "BUY",
     "deactivated_at": "2026-02-09T17:05:00",
     "deactivation_reason": "Bias changed to SELL"
   }
   ```
4. ✅ Frontend calls `/api/smart_target/activate` with `bias: SELL`
5. ✅ Backend activates new strategy:
   ```json
   {
     "strategy": "breakout_R_3_tp3.0_sl20.0",
     "product": "GBPNZD_C",
     "buy": false,
     "sell": true,
     "activated_for_bias": "SELL",
     "activated_at": "2026-02-09T17:05:00"
   }
   ```
6. ✅ Alert shows: "Deactivated 1 BUY-bias strategies, Activated breakout_R_3_tp3.0_sl20.0 (SELL bias)"

**Final State (SELL bias):**
```json
{
  "live": [
    {
      "strategy": "breakout_4_tp10.0_sl50.0",
      "product": "GBPEUR_C",
      "buy": false,
      "sell": false,
      "activated_for_bias": "BUY",
      "deactivated_at": "2026-02-09T17:05:00",
      "deactivation_reason": "Bias changed to SELL"
    },
    {
      "strategy": "breakout_R_3_tp3.0_sl20.0",
      "product": "GBPNZD_C",
      "buy": false,
      "sell": true,
      "activated_for_bias": "SELL",
      "activated_at": "2026-02-09T17:05:00"
    }
  ]
}
```
