# Plan: Mandatory Scalper Selection for AI Picker [V20260208_1400]

This plan outlines the changes to `automated_strategy_picker.py` to enforce strict scalper-only strategy selection, aligning with the `scalper_ratio` definition in `config.json`.

## 1. Requirements
*   The AI Picker (`ai_picker.html` data source) must only select strategies categorized as "Scalpers".
*   The "Scalper" definition must be dynamic, based on the `scalper_ratio` in `config.json`.
*   This strict filter applies to both CHOPPY and TRENDING market conditions.

## 2. Plan of Approach
1.  **Centralize Config Loading**: Update `automated_strategy_picker.py` to load the entire `config.json` once per scan loop.
2.  **Enforce Ratio Eligibility**: 
    *   Iterate through all strategies in `_trades_summary.json`.
    *   Extract TP and SL.
    *   Only allow strategies where `sl >= tp * scalper_ratio`. 
    *   Discard any strategy that does not meet this ratio, regardless of market momentum or status.
3.  **Update Output Metadata**: Ensure the `is_scalper` flag in the final JSON output uses the same dynamic ratio.
4.  **Verification**: Confirm that the AI Picker output only contains strategies that will be highlighted in yellow in the Performance UI.
