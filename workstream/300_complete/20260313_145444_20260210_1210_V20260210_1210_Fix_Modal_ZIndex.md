**Current Status**: COMPLETED by Claude Code (2026-02-10 12:10)
**Objective**: Fix the z-index stacking order issue where the Trade Detail Drill-Down modal appears underneath the Performance Summary Breakdown modal.

## 1. Issue Analysis
The user reported that the drill-down modal "appears underneath the performance page out of view".
The "Performance Summary Breakdown" modal (`.summary-modal`) had a z-index of **1200**.
The "Trade Detail" modal (`.trade-detail-modal`) had a z-index of **1000**.
As a result, when drilling down from the summary breakdown, the new modal was rendered behind the existing one.

## 2. Solution Implemented
Adjusted the z-indices to ensure proper stacking:
*   **`.trade-detail-modal`**: Increased from 1000 to **1300**. (Now sits above the summary modal).
*   **`.json-detail-modal`**: Increased from 1100 to **1400**. (Ensures JSON viewer sits above the trade details).
*   **`.summary-modal`**: Remains at **1200**.
*   **`.bias-notification-overlay`**: Remains at **10000** (critical alerts stay on top).

## 3. List of Changes
*   **`TradeApps/breakout/fs/strategy_performance.html`**:
    *   [x] Updated `.trade-detail-modal` z-index to 1300.
    *   [x] Updated `.json-detail-modal` z-index to 1400.
