# Plan - Table Enhancements & CSV Export

This plan covers adding the `first_seen` column to Frequency Explorer and implementing CSV export for both Strategy Performance and Frequency Explorer.

## 1. Understanding of Requirements
- **Frequency Explorer**:
    - Add `First Seen` column between `Total PNL` and `Min Seen`.
    - Add `Export to CSV` button in controls.
    - Implement `exportToCSV()` function for the current table view.
- **Strategy Performance**:
    - Add `Export to CSV` button in controls.
    - Implement `exportToCSV()` function for the main performance table.
- **Versioning**: Update to `V20260108_0030`.

## 2. Plan of Approach

### Frequency Explorer (`TradeApps\breakout\frequency_explorer.html`)
1.  **UI**:
    - Insert `<th>` for "First Seen".
    - Update table mapping logic to insert a `<td>` with the formatted `first_seen` time.
    - Add a button for "Export CSV" next to the Update button.
2.  **Logic**:
    - Update `renderTable()` to populate the new column.
    - Add `exportToCSV()` function to download the filtered and sorted data.

### Strategy Performance (`TradeApps\breakout\strategy_performance.html`)
1.  **UI**:
    - Add a button for "Export CSV" next to the Refresh button.
2.  **Logic**:
    - Implement `exportToCSV()` function to download the main table data (using `currentData` which is already calculated).

### Versioning
1.  Update `constants.py` and footer of all modified HTML files to `V20260108_0030`.
