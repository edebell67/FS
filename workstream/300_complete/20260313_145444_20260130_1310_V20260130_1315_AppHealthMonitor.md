# Plan: App Health Monitoring (Live Data & Price Feed)

This plan implements a global health check for the Breakout application.

## 1. Requirements
- Highlight all page titles in **Fluorescent RED** if:
    1. No trade files found in today's JSON folder.
    2. Price feed (`/api/vw_000_fx_quotes`) is > 5 minutes old or unavailable.

## 2. Approach
### Backend (trade_viewer_api.py)
- Create `/api/health` endpoint.
- Scan `json/live/{today}` for any `.json` files.
- Fetch `http://127.0.0.1:8001/api/vw_000_fx_quotes?db=tradedb`.
- Resolve the `time` field and check if `(Now - FeedTime) > 300s`.

### Frontend (sidebar-loader.js)
- Implement `initHealthMonitor()`.
- Poll `/api/health` every 30s.
- On error:
    - Add a global `<style>` block.
    - Set `.app-title`, `h1`, `h2` to `color: #ff0000; text-shadow: 0 0 10px #ff0000, 0 0 20px #ff0000;`.
- On success:
    - Remove the style block.

- [ ] **trade_viewer_api.py**: Add `/api/health` endpoint.
- [ ] **sidebar-loader.js**: Add polling and UI injection logic.
- [ ] **Constants.py**: Update version to `V20260130_1315`.

## 4. Verification
- Rename today's folder to test 'No files' state.
