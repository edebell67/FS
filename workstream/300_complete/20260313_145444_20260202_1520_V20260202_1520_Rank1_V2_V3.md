# Plan: Rank #1 Loader for MultiChart V2 & V3

## 1. Understanding of Requirements
The user wants to replicate the "Rank 1s" loader functionality and the "Fluorescent Yellow Dot" indicator (for when a strategy hits Rank #1) in:
*   `multi_chart_v2.html` / `multi_chart_v2.js` (SQL Version 2)
*   `multi_chart_v3.html` / `multi_chart_v3.js` (SQL Version 3)

The functionality involves:
1.  **Fetching Data**: Loading `_frequency.json` to identify when strategies hit Rank #1.
2.  **State Management**: Storing these timestamps in `firstRankOneTimes`.
3.  **Visualization**: Drawing a large yellow dot on the chart at the Rank #1 timestamp.
4.  **UI Control**: Adding a "Rank 1s" button to automatically load all such strategies.
