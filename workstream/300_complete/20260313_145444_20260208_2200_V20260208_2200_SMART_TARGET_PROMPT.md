# Plan: AI Picker "Smart Targeting" Activation & Notifications [V20260208_2200]

This plan implements a dedicated confirmation screen for the AI Picker's recommendations and a sophisticated notification management system (Mute/Suspend/Duplicate check) that updates `grid_live.json`.

## 1. Requirements

*   **Dedicated Prompt Screen**: A separate, stylish overlay/modal for confirming AI recommendations.
*   **Grid Update Logic**: If confirmed, update `grid_live.json` for the current mode.
*   **Automatic Mode**: If `ai_picker_prompt` is false in `config.json`, update `grid_live.json` immediately without the prompt.
*   **Mute Functionality**: If a notification is "muted," the action (grid update) proceeds, but the UI notification is hidden.
*   **Suspend Functionality**: Disable all new notifications/prompts entirely until resumed.
*   **Duplicate Suppression**: Ignore recommendations that correspond to a product/strategy already in `grid_live.json` or already prompted in the current session.
*   **Configuration Access**: Provide UI controls to toggle notifications on/off in `config.json`.
