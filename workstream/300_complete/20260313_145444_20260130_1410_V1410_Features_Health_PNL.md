# Plan: System Health Link and Live P&L Total

This plan adds a direct link to the System Health API in the sidebar and implements a running total P&L display for the Live Trades viewer.

## 1. Requirements
- Add a menu link in the sidebar to `http://localhost:5000/api/system_health`.
- Display a running total of the Net P&L for all live trades (open and closed) on the `live_trades.html` page.

## 2. Approach
### Sidebar (sidebar.html)
- Add a new `<a>` tag with `target="_blank"` to the sidebar menu linking to `/api/system_health`.
- Use a suitable FontAwesome icon (e.g., `fa-heartbeat`).

### Live Trades (live_trades.html)
- Add a summary card or section to the header area to show the "Total Live Net".
- Update the `loadLiveTrades()` JavaScript function to calculate the sum of `net_return` from all filtered live trades.
- Display this total with appropriate coloring (green for positive, red for negative).
