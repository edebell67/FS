# Plan for "Pure Breakout" Strategy (Strategy 14)

## 1. Goal Description
Implement "Strategy 14 - Pure Breakout" (Approach A) as requested.
Objective: Capture high-velocity price movements by entering trades when price exceeds the MIN/MAX of the last 3 price quotes.
Trigger:
- BUY if Current Price > MAX(Last 3 Prices)
- SELL if Current Price < MIN(Last 3 Prices)
Constraints:
- No open trades allowed for this strategy.
- Sequential logic (evaluate only after previous trade closes).
- Window size: Rolling 3.
Exits:
- Take Profit: +$100.00 USD (approx).
- Stop Loss: -$60.00 USD (approx).

## 2. User Review Required
- **TP/SL Configuration**: The strategy forces specific TP ($100) and SL (-$60) values via the execution call, overriding global defaults for this strategy's specific trades.
- **Buffer**: A default buffer of 0.0 is used unless configured otherwise implies "ask #" meant exact or with tiny slippage. A `pip_buffer` logic is added similar to S13 but default 0.
