# Implementation Plan: Strategy Performance DNA Breakdown Modal

This plan outlines the addition of a "DNA Breakdown" modal to the `strategy_performance.html` dashboard. Instead of a fixed section at the top, the breakdown will be accessible via a pop-up window to keep the UI clean.

## 1. Requirements
- Add a new "📊 DNA Breakdown" button to the control header, next to the "Top 20" and "CSV" buttons.
- On click, open a full-screen or centered modal displaying performance aggregations.
- Breakdown dimensions:
    - **Strategy Category** (Breakout, Breakout_R, Breakout_Rev, Breakout_R_Rev).
    - **Window Size** (2, 3, 4).
    - **TP:SL Configuration** (e.g. 10.0/20.0).
- Contents per card:
    - Total Trades, #Buys, #Sells.
    - Total Net, Buy Net, Sell Net.
    - Win Rate % (calculated as count of trades with net > 0).

## 2. Proposed Changes

### 2.1 CSS Integration (`strategy_performance.html`)
- Add styles for `#breakdownModal` and its content.
- Reuse premium glassmorphism styling from existing modals.
- Define `.breakdown-grid` for layout inside the modal.

### 2.2 HTML Structure (`strategy_performance.html`)
- Add `<button id="breakdownBtn" class="btn-breakdown">📊 DNA Breakdown</button>` to the control group.
- Add `<div id="breakdownModal" class="modal">...</div>` at the bottom of the body.

### 2.3 JavaScript Logic (`strategy_performance.html`)
- **Aggregation Function**: `calculateDNABreakdown()`
    - Iterates through `currentData` (the data currently loaded in the dashboard).
    - Uses Regex to extract Window Size and TP:SL.
    - Computes totals for each bucket.
- **Render Function**: `showDNABreakdown()`
    - Calls `calculateDNABreakdown()`.
    - Generates the HTML for the modal.
    - Displays the modal.
