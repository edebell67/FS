### **2. Auto-Deactivation Logic**

**Create new API endpoint:** `/api/deactivate_wrong_bias`

**Logic:**
```python
def deactivate_wrong_bias(current_bias):
    opposite_bias = 'SELL' if current_bias == 'BUY' else 'BUY'
    
    # Load grid_live.json
    activations = load_grid_live()
    
    deactivated = []
    for activation in activations:
        if activation.get('activated_for_bias') == opposite_bias:
            # Deactivate this strategy
            activation['buy'] = False
            activation['sell'] = False
            deactivated.append(activation)
    
    # Save updated grid_live.json
    save_grid_live(activations)
    
    return deactivated
```

### **3. Bias-Flip Handler Enhancement**

**Update frontend code:**
```javascript
async function onBiasFlip(oldBias, newBias, recommendedStrategy) {
    // 1. Deactivate wrong-bias strategies
    const deactivateResp = await fetch('/api/deactivate_wrong_bias', {
        method: 'POST',
        body: JSON.stringify({current_bias: newBias})
    });
    
    const result = await deactivateResp.json();
    console.log(`Deactivated ${result.count} wrong-bias strategies`);
    
    // 2. Activate new strategy with bias tag
    const activateResp = await fetch('/api/activate_strategy', {
        method: 'POST',
        body: JSON.stringify({
            strategy: recommendedStrategy.strategy,
            product: recommendedStrategy.product,
            bias: newBias  // Tag with current bias
        })
    });
    
    // 3. Show confirmation
    alert(`Switched from ${oldBias} to ${newBias} bias`);
}
```
