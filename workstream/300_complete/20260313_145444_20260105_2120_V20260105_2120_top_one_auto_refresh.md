# Plan: Top One Auto Refresh (2026-01-05)

This plan outlines the implementation of an auto-refresh feature for the "Top One Performer" viewer.

## 1. Understanding of Requirements
The user wants the "Top One Performer" screen to automatically refresh its data periodically without requiring manual clicks on the "Update View" button.

## 2. Plan of Approach
1.  **UI Enhancement**: 
    *   Add a checkbox labeled "Auto Refresh" next to the "Update View" button in `top_one_viewer.html`.
    *   Set a default interval of 30 seconds.
2.  **JavaScript Implementation**:
    *   Create a variable to hold the `setInterval` timer ID.
    *   Implement a `toggleAutoRefresh()` function that starts or stops the timer based on the checkbox state.
    *   Modify `fetchData()` to ensure it doesn't cause issues if called while an auto-refresh is already pending (though unlikely to be an issue with standard fetch).
    *   Initialize auto-refresh on page load if desired (will default to 'off' for safety).
3.  **Version Update**:
    *   Update `constants.py` with the new version string: `V20260105_2120`.
    *   Update the hardcoded version in the `top_one_viewer.html` footer.
