### **Step 3: Strategy Promotes to Live** ⚠️ KEY BOTTLENECK
**File**: `common.py` → `_create_l_trade_order()` (line 2255)
**Action**: The strategy script calls this function to:
1. Check if strategy/product is in `grid_live.json` ✅
2. Check `max_live_trades` limit (line 2279-2288) ✅
3. Count current live trades via `_get_total_open_live_trades()` ✅
4. Create L-Trade JSON in `C:\Users\edebe\eds\trades_rt3\orders\` ✅

**Guard Logic** (lines 2279-2288):
```python
if not is_close:
    max_live_trades_global = int(_config.get('max_live_trades', 1))
    json_base_dir = os.path.join(os.path.dirname(CONFIG_FILE_PATH), 'json', run_mode.lower())
    current_live_count = _get_total_open_live_trades(json_base_dir)
    
    if current_live_count >= max_live_trades_global:
        print(f"[GUARD] Max live trades ({max_live_trades_global}) reached. Currently have {current_live_count} open. Skipping L-Trade order...")
        return None
```

**This is where L-Trade OPEN orders are created!**
