# Plan: Frequency Explorer Drill-Down Feature

**Version:** V20260130_0915
**Date:** 2026-01-30 09:15

## 1. Understanding of Requirements
The user wants to add a drill-down capability to the Frequency Explorer. Clicking on a "Product" name in the leaderboard or consistency list should open a modal showing the history of that specific product-strategy combination across all snapshots (from start of day to now).

The drill-down should display:
- Snapshot Time
- Net P&L at that time
- Rank (Highlighting if #1 or Top 5)
- Score contribution for that snapshot

## 2. Plan of Approach
1.  **Modal UI**: Add a sleek, glassmorphism-style modal to `frequency_explorer.html`.
2.  **Make Product Clickable**: Update the `renderTable` and `renderConsistencyList` functions to wrap the product name in a clickable element (anchor or span with cursor: pointer).
3.  **Data Processing**: Implement `showDrillDown(product, strategy)` which iterates through `rawData.snapshots` in reverse chronological order and extracts the relevant stats for the selected pair.
4.  **Display Logic**: Render a table inside the modal with the snapshot history.
5.  **Clean Close**: Implement standard close functionality (button + clicking backdrop).
