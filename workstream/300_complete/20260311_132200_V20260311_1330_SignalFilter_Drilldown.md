# Plan: Add Signal Filter to Trade Drilldown

**Version:** V20260311_1330
**Date:** 2026-03-11 13:22
**Description:** Add a signal filter (All/Long/Short) to the Trade Drilldown screen in the Multi-Chart P&L Grid.

## 1. Requirements
The user wants to filter the trades shown in the drilldown modal by their signal (LONG/SHORT).
- Component: `tradeDrilldownModal`
- Options: `All` (default), `Long`, `Short`
- Behavior: Update the table display immediately when the filter changes.

## 2. Proposed Changes

### 2.1 `fs/multi_chart.html`
- Insert a new `field-group` or similar wrapper next to the existing `drilldownModelSelect` dropdown.
- Add a `<select>` with id `drilldownSignalSelect`.
- Options: `All`, `Long`, `Short`.

### 2.2 `fs/multi_chart.js`
- Add a global state variable `drilldownSignalFilter` initialized to `'all'`.
- In `initEventHandlers`, register a change listener for `drilldownSignalSelect`.
- In `showTradeDrilldown`:
    - Reset `drilldownSignalFilter` to `'all'`.
    - Reset the dropdown UI value to `'all'`.
- In `renderDrilldownTable`:
    - Incorporate `drilldownSignalFilter` into the filtering logic applied to `currentDrilldownTrades`.

### 2.3 `fs/constants.py`
- Update `VERSION` to `"V20260311_1330"`.
