# Top 20: Directional Auto-Activation (2026-02-05)

This plan outlines the enhancement of the "Auto" batch activation feature in the Top 20 screen. It allows users to choose whether to activate Buy (Long), Sell (Short), or Both directions for a top-performing strategy.

## 1. Understanding of Requirements
Currently, the "Auto" button in the Top 20 modal activates both 'long' and 'short' directions simultaneously. The user wants the granularity to activate only one direction if desired, while still retaining the 'both' option.

## 2. Plan of Approach
1.  **Refactor JavaScript Logic**:
    *   Update `syncActivationFromTop20(strategyKey, product)` to `syncActivationFromTop20(strategyKey, product, direction)`.
    *   Modify the function body to conditionally call `toggleAutoActivation` based on the `direction` parameter ('long', 'short', or 'both').
2.  **Enhance UI Buttons**:
    *   In the `renderTop20Table` function, replace the single "Auto" button with a cluster of three compact buttons:
        *   **B (Buy)**: Triggers 'long' activation. Styled with emerald green accents.
        *   **S (Sell)**: Triggers 'short' activation. Styled with rose red accents.
        *   **Both**: Triggers both directions. Styled with amber/gold accents (original look).
    *   Add descriptive tooltips to each button to clarify the action.
3.  **Versioning**: Update `Constants.py`.
