# 2026-02-15 V20260215_1440 Collapse Live Trades Sections

## 1. Understanding of Requirements
The user wants to organize the `live_trades.html` view by "collapsing each section" so only the summary is visible initially.
Each "Open Positions" and "Closed Executions" section should be expandable via buttons.
This implies grouping the Summary, Open, and Closed entries for each strategy/bucket into a coherent unit, with the details hidden by default.

## 2. Plan of Approach
1.  **Modify `renderTrades` in `live_trades.html`**:
    *   Create a wrapper `div` for each group (Strategy/Bucket) to hold Summary, Open, and Closed sections together.
    *   Generate unique IDs for the "Open" and "Closed" sections.
    *   Pass these IDs to the `createGroupSummary` and `createTableSection` functions.
2.  **Modify `createGroupSummary`**:
    *   Add "Expand/Collapse" buttons for Open and Closed sections.
    *   These buttons will toggle the visibility of the respective sections.
3.  **Modify `createTableSection`**:
    *   Accept an `id` and an `isHidden` (boolean) parameter.
    *   Set `display: none` if `isHidden` is true.
4.  **Add `toggleView` Helper**:
    *   A simple function to toggle `display` between `none` and `block`.

## 3. List of Changes
