# Plan: Rank Improvement Alert System (2026-01-30)

**Objective**: Extend the Rank Alert functionality to notify the user not only when a strategy hits Rank #1, but also when it demonstrates significant rank improvement (e.g., entering Top 5 or climbing within the Top 5).

## 1. Requirements
- [ ] **Track Rank History**: Maintain a record of previous rankings for all active strategies.
- [ ] **Detect Improvement**: Identify when a strategy's rank index decreases (moves up the leaderboard).
- [ ] **Threshold Logic**: 
    - Trigger Alert if: 
        - Strategy enters Top 5 from outside.
        - Strategy climbs within Top 5 (e.g., #4 -> #2).
    - Ignore minor fluctuations outside Top 5.
- [ ] **Generalized User Interface**: Update the "Rank #1" yellow box to a generic "Rank Alert" box.
    - Title: "Rank Improved!" vs "New Rank #1!".
    - Body: Show "Rank Change: #Old -> #New".
- [ ] **Generalized Actions**: Ensure "Monitor Only", "Monitor Priority", etc., apply to the *subject* of the alert.
- [ ] **Configurability**: Add settings to enable/disable "Improvement Alerts" separately from "Rank #1 Alerts".

## 2. Approach

### frontend (frequency_explorer.html)
1.  **State Management**:
    - Add `let previousRanks = {};` (Map: `product|strategy` -> `rankIndex`).
    - Add `let rankImprovementAlertEnabled = true;`.

2.  **Detection Logic (`checkRankChanges`)**:
    - Iterate `leaders` list (which is sorted by score).
    - For each item `i` (where `i` is current rank):
        - Get `key`.
        - Get `oldRank = previousRanks[key]`.
        - If `oldRank` exists:
            - If `i < 5` (Current Top 5) AND `i < oldRank`:
                - Trigger Alert (Improvement).
        - Update `previousRanks[key] = i`.
    - Special Case: Keep `checkRank1Change` logic or merge? Merge is better. "Rank #1" is just `i === 0` and `oldRank > 0`.

3.  **UI Updates (`showRankAlert`)**:
    - Accept `alertType` ('rank1', 'improvement').
    - If `improvement`:
        - Title: `<i class="fas fa-arrow-up"></i> Rank Improvement!`
        - Subtext: `Rank: #${oldRank+1} ➔ #${newRank+1}`.
        - Color: Maybe slightly different yellow or Green? Keep Yellow for consistency/urgency.
    - If `rank1`:
        - Title: `<i class="fas fa-trophy"></i> New Rank #1!`

4.  **Config**:
    - Add checkbox "Alert on Rank Improvement" in the footer or settings.
