# Plan: Automated Trade Source Mutual Exclusion

This plan outlines the implementation of a new `automated_trade_source` field in `config.json` to ensure that `grid_live.json` only receives automated updates from one source (Frequency, Trade Bucket, or Top20) at a time. The source is locked whenever a manual trigger occurs for one of these functionalities.

## 1. Understanding of Requirements
- Add `automated_trade_source` to `config.json`.
- Possible values: `Frequency`, `Trade Bucket`, `Top20`.
- Manual triggers for any of these must update the field to that source.
- Automated updates to `grid_live.json` must be blocked if their source does not match the current `automated_trade_source`.

## 2. Plan of Approach

### A. Modify `config.json`
- Initialize `automated_trade_source` in `config.json` (if missing).

### B. Modify `trade_viewer_api.py`
1.  **Helper Functions**:
    - `_update_automated_source(source_name)`: Updates `config.json` with the new source.
    - `_is_source_allowed(requested_source)`: Returns `True` if `requested_source` matches the current `automated_trade_source` or if it's a manual UI override.
2.  **Top20 (push_to_grid_v2)**:
    - On call, set `automated_trade_source` to `Top20`.
    - Check if the source matches before applying changes.
3.  **Trade Bucket (update_trade_bucket & sync functions)**:
    - In `update_trade_bucket`, if `live` is toggled ON, set `automated_trade_source` to `Trade Bucket`.
    - In `_sync_bucket_to_grid_live` and `_reconcile_active_buckets`, check if the source matches before applying changes to `grid_live.json`.
4.  **Frequency (update_grid_live & update_config)**:
    - In `update_config`, if Frequency Notifier (`rank_alert_suspended` -> `False`) is re-enabled, set `automated_trade_source` to `Frequency`.
    - In `update_grid_live`, if `source` is `rank_alert_ui` (manual trigger), set `automated_trade_source` to `Frequency`.
    - In `update_grid_live`, if `source` is `rank_alert_auto` (automated trigger), block if it doesn't match `Frequency`.
5.  **General Source Management**:
    - Update `api/config` POST (update_config) to allow updating this field if needed from UI.
    - Ensure `ui` manual clicks on charts are either allowed or handled as overrides.

### C. Modify `Constants.py`
- Update version number to `V20260205_2100`.
