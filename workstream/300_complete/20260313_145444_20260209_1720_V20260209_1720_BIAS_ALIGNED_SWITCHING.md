**Version:** V20260209_1720

## 🎯 **CORE PRINCIPLE**

**"Always be in a strategy that matches the current market bias"**

- **BUY bias** → Only hold BUY-favoring strategies
- **SELL bias** → Only hold SELL-favoring strategies
- **Bias flip** → Immediately switch strategies

**Example:**

  "activated_for_bias": "BUY",  // NEW FIELD

### **Enhancement 2: Auto-Deactivate Wrong-Bias Strategies**

**When bias flips from BUY → SELL:**
1. Scan `grid_live.json`
2. Find all strategies with `activated_for_bias: "BUY"`
3. **Auto-deactivate** those strategies
4. **Log the deactivation** with reason: "Bias changed to SELL"

**When bias flips from SELL → BUY:**
1. Scan `grid_live.json`
2. Find all strategies with `activated_for_bias: "SELL"`
3. **Auto-deactivate** those strategies
4. **Log the deactivation** with reason: "Bias changed to BUY"

### **Enhancement 3: Bias-Flip Action Handler**

**When user clicks "Activate Strategy" in bias-flip modal:**
1. ✅ Activate recommended strategy
2. ✅ Tag it with `activated_for_bias: <NEW_BIAS>`
3. ✅ **Auto-deactivate** all strategies tagged with opposite bias
4. ✅ Log the action

**When user clicks "Auto-Activate (No More Prompts)":**
1. ✅ Set `ai_picker_prompt: false`
2. ✅ On future bias flips:
   - Auto-activate top recommended strategy
   - Tag with current bias
   - Auto-deactivate opposite-bias strategies
   - No modal popup

## 🚨 **EDGE CASES TO HANDLE**

### **1. Manual Activations**
**Question:** Should manually activated strategies be auto-deactivated on bias flip?

**Options:**
- **A)** Respect manual activations (don't auto-deactivate)
- **B)** Auto-deactivate all, including manual
- **C)** Prompt user: "Bias flipped. Deactivate manual strategies?"

**Recommendation:** **Option A** - Respect manual activations, but warn user

### **2. Multiple Active Strategies**
**Question:** Can multiple strategies be active for the same bias?

**Recommendation:** **Yes** - Allow multiple, but all must match current bias

### **3. Bias Status: MIXED**
**Question:** What to do when status is MIXED (day bias ≠ recent bias)?

**Recommendation:** **Use day bias** - Don't flip until day bias actually changes

### **4. Rapid Bias Flips**
**Question:** What if bias flips multiple times in short period?

**Recommendation:** 
- Add **cooldown period** (e.g., 5 minutes)
- Don't trigger flip if last flip was < 5 min ago
- Log: "Bias flip ignored - cooldown active"
