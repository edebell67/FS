# Plan: Add Direction (Long/Short) Filter to Performance Breakdowns [V20260215_2100]

This plan outlines the addition of a Long/Short (Direction) filter to both the Performance Summary Breakdown (Summary Modal) and the Trade Drill-Down (Trade Modal).

## 1. Understanding of Requirements
*   **Objective**: Allow users to filter performance data by Buy (Long) or Sell (Short) trades.
*   **Scope**: 
    1.  **Main Performance Table Drill-Down (`tradeModal`)**: Add a dropdown to switch between "All", "Buy", and "Sell" trades while viewing details. This allows users who clicked "Total Net" to filter by direction, and users who clicked "Buy Net" to switch to "Sell" without closing the modal.
    2.  **Performance Summary Breakdown (`summaryModal`)**: The "Best Parameters" screen already has metric toggles. We will ensure these are functioning as intended and potentially add a way to filter the hierarchy view as well.

## 2. Plan of Approach

### A. Trade Drill-Down (`tradeModal`)
1.  **UI Update**: Add a `drilldownDirectionSelect` dropdown next to the `drilldownModelSelect` in the `modal-header`.
2.  **Logic Update**:
    *   **Fetch Adjustment**: Modify the API call in `showDrillDown` to fetch `direction=all` trades when a specific product/strategy/parm is selected. This enables switched-on-the-fly filtering.
    *   **State Management**: Store the user's initial direction request to pre-select the dropdown.
    *   **Filtering**: Update `renderDrillDownTable` to filter the `currentDrillDownTrades` by both the selected Model and the selected Direction.
3.  **CSV Export**: Update `exportDrillDownToCSV` to respect the active filters in the modal.

### B. Summary Modal (`summaryModal`)
1.  **Metric Toggles**: Ensure the existing "Total / Buy / Sell" buttons in the "Best Parameters" panel are highly visible and working.
2.  **Global Toggle (Optional/Extension)**: If required, add a global direction filter to the summary modal header that affects both the hierarchy and leaderboard.
