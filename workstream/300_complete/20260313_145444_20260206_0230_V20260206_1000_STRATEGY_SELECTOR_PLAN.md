# Strategy Promotion Agent Implementation (2026-02-06)

This plan outlines the creation of an automated agent that selects the best-performing trading strategies from daily summary data and promotes them to the live trading grid.

## 1. Understanding of Requirements
- **Input**: `C:\Users\edebe\eds\TradeApps\breakout\fs\json\live\2026-02-06\_summary_net.json` (or the processed `_top20.json`).
- **Goal**: Automatically select the top strategy/product combination to trade.
- **Target**: Update `grid_live.json` with the selection.
- **Audit**: Log changes in `grid_live_history.json`.

## 2. Plan of Approach
1.  **Script Creation**: Create `automated_strategy_selector.py`.
2.  **Selection Logic**:
    - Load `_top20.json` for the current date.
    - Select the #1 ranked strategy by `total_net`.
    - Ensure it has positive P&L (`total_net > 0`) and has executed trades (`trade_count >= 1`).
3.  **Grid Promotion**:
    - Load current `grid_live.json`.
    - Upsert the selected strategy into the `live` list.
    - Assign `source` as `auto_selector_{YYYY-MM-DD}`.
4.  **History Logging**: Append the new grid state to `grid_live_history.json` with a timestamp.
5.  **Versioning**: Update system version to `V20260206_1000`.
