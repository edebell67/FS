# Plan: Card Management - Reordering & Inter-View Transfer (V20260128_2135)
**Date:** 2026-01-28 21:35

## 1. Requirements Understanding
- Provide functionality to reorder cards within the same view (Move Left/Right).
- Provide functionality to transfer a card and its nested strategies from one saved view (Layout Preset) to another.

## 2. Approach
- **Reordering:**
    - Add `chevron-left` and `chevron-right` buttons to the `chart-card` header.
    - Implement `moveGroup(groupName, direction)` which swaps the group's position in the `activeOverlays` list.
- **Inter-View Transfer:**
    - Add an `exchange-alt` button to the `chart-card` header.
    - Create a new modal `transferCardModal` in `multi_chart.html`.
    - Implement `openTransferModal(groupName)` to populate the target view list from `localStorage`.
    - Implement `submitTransfer()` which:
        1. Extracts all overlays for the selected group.
        2. Appends them to the target preset in `localStorage`.
        3. Removes them from the current display (`activeOverlays`).
        4. Saves the target preset and refreshes the chart grid.
