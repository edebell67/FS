## 5. Implementation Details

### Helper Function: count_current_live_trades()
```python
def count_current_live_trades(date_str, mode='live'):
    """Count all currently open trades marked as live."""
    count = 0
    date_folder = JSON_TRADES_BASE / mode / date_str
    if not date_folder.exists():
        return 0
    
    for op_file in date_folder.glob("*_op.json"):
        try:
            with open(op_file, 'r') as f:
                data = json.load(f)
            if data.get('is_live_trade') or data.get('order_sent_net') or data.get('order_sent_alt'):
                count += 1
        except:
            pass
    return count
```
