# Plan: Generate _live_trades.json Script

## 1. Understanding of Requirements
The user wants a Python script to:
1.  Scan the `fs/json/live/{date}` directory (specifically 2026-02-02, but should be generic).
2.  Identify all trade files (`*.json`) where `is_live_trade` is `true`.
3.  Consolidate these trades into a single output file named `_live_trades.json`.
4.  Save this file in the same directory.

## 2. Plan of Approach
1.  **Script Name**: `live_trades_generator.py`.
2.  **Location**: `C:\Users\edebe\eds\TradeApps\breakout\`.
3.  **Functionality**:
    *   Accept optional command line arguments for `mode` (default: 'live') and `date` (default: current date).
    *   Construct the target directory path.
    *   Iterate through all JSON files in the directory.
    *   Filter out metadata files (starting with `_`).
    *   Parse each file:
        *   Check for `is_live_trade: true` (or string 'true').
        *   Check for `live_trade_executed: true` (as a fallback/safety).
    *   Collect matching trade data.
    *   Sort trades by `entry_time` (descending).
    *   Write the list to `_live_trades.json`.
    *   Add metadata to the output (generation time, count, session max net from summary).

## 3. Script Structure
```python
import json
import os
import argparse
from datetime import datetime
from pathlib import Path

# Paths
BASE_DIR = Path(r"C:\Users\edebe\eds\TradeApps\breakout\fs\json")

def generate_live_trades(mode, date_str):
    target_dir = BASE_DIR / mode / date_str
    if not target_dir.exists():
        print(f"Directory not found: {target_dir}")
        return

    live_trades = []
    
    # helper to check truthy
    def is_true(val):
        return val is True or str(val).lower() == 'true'

    print(f"Scanning {target_dir} for live trades...")
    
    files = list(target_dir.glob("*.json"))
    for file_path in files:
        if file_path.name.startswith("_"): continue

        try:
            with open(file_path, 'r') as f:
                data = json.load(f)
                
            # Check conditions
            if is_true(data.get('is_live_trade')) or \
               is_true(data.get('live_trade_executed')) or \
               is_true(data.get('order_sent_net')):
                
                # Enrich with filename for reference
                data['_filename'] = file_path.name
                live_trades.append(data)
                
        except Exception as e:
            print(f"Error reading {file_path.name}: {e}")

    # Sort
    live_trades.sort(key=lambda x: x.get('entry_time', ''), reverse=True)

    # Output Structure
    output = {
        "generated_at": datetime.now().isoformat(),
        "mode": mode,
        "date": date_str,
        "count": len(live_trades),
        "trades": live_trades
    }

    # Write to file
    outfile = target_dir / "_live_trades.json"
    with open(outfile, 'w') as f:
        json.dump(output, f, indent=4)
        
    print(f"Successfully wrote {len(live_trades)} trades to {outfile}")

# ... (main block)
```
