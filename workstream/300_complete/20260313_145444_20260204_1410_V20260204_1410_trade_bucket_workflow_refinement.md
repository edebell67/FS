- **Archiving**: Append current `grid_live.json` state to `json/grid_live_history.json` with a timestamp before any modification.
- **Trade Bucket Promotion**: 
    - Replace `activations_is_live.json` sync with `grid_live.json` sync.
    - Promotes only the "Leader" (Top Performer) of the bucket.
    - Format: `{"model": "...", "product": "...", "metric": "net", "group": "{bucket_name}", "source": "trade_bucket:{bucket_name}", "activated_at": "..."}`.
- **Mutual Exclusion**:

        - Reject incoming `grid_live` updates from `rank_alert` sources.
    - If Frequency Notifier is set to ACTIVE (`rank_alert_suspended: false`):
        - Deactivate all Trade Buckets.
- **Consistency**: Update version in `fs/constants.py` and `db/constants.py` to `V20260204_1410`.
