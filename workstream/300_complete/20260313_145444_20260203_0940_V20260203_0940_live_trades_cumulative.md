# Live Trades Cumulative Net & Daily Target Highlight

## 1. Understanding of Requirements
The user wants to enhance the `live_trades.html` viewer with:
1.  A **Cumulative Net** column in the table, showing the running total of P&L.
2.  A **Row Highlight** feature that visually marks the specific trade where the cumulative P&L first crosses the daily target (defined in `config.json`).

## 2. Plan of Approach
1.  **Config Loading**:
    *   Update `loadConfig()` in `live_trades.html` to extract `daily_target`.
    *   Handle the API response structure `{ success: true, config: {...} }` correctly.
2.  **Cumulative Calculation**:
    *   In `loadLiveTrades()`, step through the chronologically sorted trades.
    *   Maintain a running total (`cumulative_net`).
    *   Check for the crossover event: `if (previous_total < daily_target && current_total >= daily_target)`.
    *   Mark the specific trade with a `target_met` flag.
3.  **UI Rendering**:
    *   Update `createTableSection()` to include the "Cum. Net" column.
    *   Apply a CSS class `.target-met-row` to the row where `target_met` is true.
    *   Add CSS styles for the highlight.
