[V20260205_0955] Add Model Filter to Trade Lists Across All Views

## 1. Requirement
The user wants to add a "Model" search filter to all trade lists. This includes:
1.  **Multi-Chart Drilldown Modal**: Currently shows trades for multiple strategies in a card but lacks a way to isolate one.
2.  **Live Trades View (`live_trades.html`)**: The main list of active/closed trades for the day.
3.  **Strategy Performance Drilldown**: The modal used in `strategy_performance.html`.

## 2. Plan of Approach

### A. Multi-Chart Drilldown (`multi_chart.html` / `multi_chart.js`)
1.  **UI**: Replace text input with a professional `<select>` dropdown in the `modal-header` or a dedicated filter bar.
2.  **Logic**: 
    *   Dynamically populate the `select` with unique models from the trade data.
    *   Show "All Models" by default.
    *   Update `renderDrilldownTable` to filter trades based on the selection.

### B. Live Trades View (`live_trades.html`)
1.  **UI**: Add a themed `<select>` for "Filter Strategy" (Models).
2.  **Logic**: Dynamically populate unique strategies from `currentTrades`.

### C. Strategy Performance Drilldown (`strategy_performance.html`)
1.  **UI**: Add a selection list in the modal header.
2.  **Logic**: Repopulate on each drill-down open.
