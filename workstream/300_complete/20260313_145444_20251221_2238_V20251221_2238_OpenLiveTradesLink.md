# Plan: Show Open Live (L) Trades Count and Link to Underlying Trades

This document outlines the plan to update the "Open Trades" card in the Breakout Trade Viewer to display the count of open "Live" trades and provide a link/click action to view those specific trades.

## 1. Understanding of Requirements
- The "Open Trades" card currently shows only the total number of open trades.
- Requirement: Add a sub-count for "Live" (L) trades that are open.
- Requirement: Make these counts interactive (clickable) to open the trade details modal showing the respective trades.

## 2. Plan of Approach
The changes will be primarily in `trade_viewer.html` and the version constant in `constants.py`.

1.  **Refactor Table Rendering**: Extract the trade table rendering logic from `showFilteredTrades` into a more generic function `renderTradesToModal` to allow reuse for global filters.
2.  **Update `renderStats`**:
    - Calculate `openLiveTradesCount` by filtering `trades` for `status === 'OPEN'` and `is_live_trade === true`.
    - Update the HTML for the "Open Trades" card to show both total open and live open counts.
    - Add `onclick` handlers to the counts to trigger the modal view.
3.  **Add `showOpenTrades(liveOnly)`**: A new helper function that filters all trades and calls the refactored modal renderer.
4.  **Update Version**: Update `VERSION` in `constants.py`.
