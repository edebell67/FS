# Plan: Add Activation Removal Feature (v20260501_0025)

## 1. Objective
Add a "Remove" button to each strategy card in the Activation Explorer UI to allow permanent deletion of activation keys from `activations.json`.

## 2. Plan of Approach

2.  **Frontend (activations_explorer.html)**:
    - Add CSS for a red "Remove" button.
    - Update `renderCards()` to inject the button into each card.
    - Implement `removeActivation(key)` JavaScript function to call the backend with a confirmation prompt.
3.  **Consistency**:
    - Update `VERSION` in `constants.py`.
    - Update hardcoded version in `activations_explorer.html`.
