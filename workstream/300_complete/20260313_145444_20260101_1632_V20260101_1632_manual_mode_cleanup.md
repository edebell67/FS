# Plan: Automatic Deactivation of Automated Strategies in Manual Mode

This document outlines the plan to modify the backend logic to ensure that any "automated" strategies (`manual: false`) are automatically deactivated when their activation mode is switched to `manual`.

## 1. Understanding of Requirements

*   **Trigger**: Switching `Net Activation Mode` or `Alt Activation Mode` from `auto` to `manual` in `config.json`.
*   **Action**: The backend process (`_perform_auto_activation_check`) should identify all activation records in `activations.json` that are:
    1.  Marked as `manual: false` (i.e., they were created by the system).
    2.  Belong to a mode that is now set to `manual`.
    *   These records should be set to `active: false` (deactivated).
*   **Goal**: Prevent "orphaned" automated trades from continuing to fire when the user has explicitly requested manual control.

## 2. Plan of Approach

1.  **Modify `common.py`**:
    *   Update `_perform_auto_activation_check`.
    *   Refactor the deactivation loop to handle the "Manual Mode Cleanup" scenario.
    *   If `auto_net` is `False` (Manual Mode), ensure all `_net` keys with `manual: false` are added to the deactivation list.
    *   If `auto_alt` is `False` (Manual Mode), ensure all `_alt` keys with `manual: false` are added to the deactivation list.
3.  **Modify `trade_viewer.html`**:
    *   Update `saveConfig` to detect if `net_activation_mode` or `alt_activation_mode` is changing from `auto` to `manual`.
    *   If detected, show a specific confirmation prompt: *"Switching to MANUAL mode will automatically DEACTIVATE all system-managed (automated) strategies. Do you want to continue?"*

4.  **Modify `trade_viewer.html`**:
    *   Add `.auto-selected` CSS class (darker green background).
    *   Update `renderStrategyCard` to apply `.auto-selected` to direction groups if they are auto-activated (`active: true` and `manual: false`).
    *   Change the "Net" and "Alt" labels or buttons to use this darker green color when auto-selected.

5.  **Update Version**:
    *   Increment the version in `constants.py` to `V20260101_1950`.
