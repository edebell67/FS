# Plan - Filter Summary Generator to Current Mode and Date

This plan outlines the changes to `summary_net_generator.py` to only process the current trading mode and date, as requested.

## 1. Understanding of Requirements
- The user wants the `_summary_net.json` for 2026-01-23 corrected.
- The generator script should be narrowed down to only process the `run_mode` specified in `config.json`.
- The generator script should only process the current date (today).

2.  **Update `Constants.py`**:

3.  **Execution**:
