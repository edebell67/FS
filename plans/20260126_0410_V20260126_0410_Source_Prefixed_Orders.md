# Plan: Screen-Source Prefixed Order Filenames [V20260126_0410]

This plan implements the user's request to modify the naming convention of tradeable JSON order files sent to TWS so that they start with the name of the source screen (e.g., `grid_live_`, `trade_bucket_`, or `breakout_`).

## 1. Requirements
*   JSON order files in the TWS orders directory must be prefixed with their source screen.
*   Identify sources: `grid_live_monitor.py` (Grid screen), `trade_bucket` (Trade Buckets screen via `activations_is_live.json`), and general background `breakout` engine.
