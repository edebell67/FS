# Plan - Top One Frequency Tracker (V20260103_0430)

This plan outlines the creation of a frequency monitoring script to track strategy leadership patterns.

## 1. Understanding of Requirements
- Monitor `_top_one.json` every minute.
- Maintain a tally of how often a specific `(product, strategy)` pair is at the #1 position.
- Categories: `buy`, `sell`, and `combo`.
- Output: A persistent JSON file named `top_one_frequency.json`.

## 2. Logic Details
- **Dynamic Path**: `{ROOT}/json/{mode}/{date}/_top_one.json`.
- **Combo Logic**: Compare `total_net_return` of `top_buy` and `top_sell` from the leader file. The winner is recorded as the `combo` winner for that interval.
- **Persistence**: Load `top_one_frequency.json` at start, update in memory, and save every minute.

## 3. List of Changes
- **`top_one_frequency.py`**: New standalone monitoring script.
- **`constants.py`**: Version bump.

## 4. Verification
- Verify the script correctly identifies the leaders and increments the #1 count in the JSON file.
