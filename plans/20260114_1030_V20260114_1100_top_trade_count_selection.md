# Add Top Trade Count V-Trade Selection Mode

## Goal Description

Add a new virtual trade selection mode `top_trade_count` that selects the top buy/sell leader based on the number of trades executed, rather than PnL or frequency. This provides a more stable selection criteria that's less susceptible to volatility from individual high-profit trades.

When multiple strategies have the same trade count, the system will use total PnL by product as a tiebreaker.

## User Review Required

> [!IMPORTANT]
> This change adds a third selection mode but does not modify existing `top_net` or `top_frequency` modes. The user can switch between all three modes via the `vtrade_selection_type` config setting.
