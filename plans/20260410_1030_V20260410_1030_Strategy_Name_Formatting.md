# Plan: Strategy Name Formatting for Social Posting

## 1. Understanding of Requirements
The goal is to refine the strategy name display in the consolidated social posting leaderboard. Specifically, the stop loss parameter (`slN`) should be removed from the raw strategy name string. 

Example:
- Current: `brk R 2 tp20 sl5`
- Requested: `brk R 2 tp20`

This change applies to the "relevant strategy name" (the shortened raw parameters), not the strategy alias (derived name).

## 2. Plan of Approach
1.  **Update Version**: Change `VERSION` in `TradeApps/breakout/fs/constants.py` to `V20260410_1030`.
2.  **Modify `shorten_strategy_name`**: Update the function in `generate_posting_package.py` to remove the `sl` part.
3.  **Validation**: Run the generator and verify the output.
