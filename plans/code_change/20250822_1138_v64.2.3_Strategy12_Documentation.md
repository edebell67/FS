# Strategy Documentation: Strategy 12

- **Version:** 64.2.5
- **Date:** 2025-08-22
- **Strategy Name:** Strategy 12
- **Strategy ID:** Strategy12

## Description

Executes trades based on a comparison of closed trade counts from the prior block. It uses the global 'Return Type' UI selector to determine the logic for trade direction based on which side (BUY or SELL) had superior performance in the last block.

## UI Configuration

This strategy uses the global "Return Type" dropdown menu in the main UI.

## Buy Criteria

A BUY signal is triggered and executed if one of the following conditions is met:

### Condition A
- **Return Type:** `alt_net_return`
- **AND** Total closed SELL trades > Total closed BUY trades
- **AND** Positive closed SELL trades > Positive closed BUY trades

**OR**

### Condition B
- **Return Type:** `net_return`
- **AND** Total closed BUY trades > Total closed SELL trades
- **AND** Positive closed BUY trades > Positive closed SELL trades

### Action on Buy Signal
- Close all currently open SELL trades.
- Execute a new BUY trade.

---

## Sell Criteria

A SELL signal is triggered and executed if one of the following conditions is met:

### Condition C
- **Return Type:** `alt_net_return`
- **AND** Total closed BUY trades > Total closed SELL trades
- **AND** Positive closed BUY trades > Positive closed SELL trades

**OR**

### Condition D
- **Return Type:** `net_return`
- **AND** Total closed SELL trades > Total closed BUY trades
- **AND** Positive closed SELL trades > Positive closed BUY trades

### Action on Sell Signal
- Close all currently open BUY trades.
- Execute a new SELL trade.
