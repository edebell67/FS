# Plan: Strategy 02 - Top Performer Immediate Execution

This document outlines the plan to create the `02_str_topPerformerCopy.py` strategy. This strategy will be a modification of the existing "Top Performer" strategy.

## 1. Requirements

The new strategy will:
1.  Be based on the `01_str_topPerformer.py` script.
2.  Use the same API endpoint (`vw_106_ModelPerformance_alt`) to find the single best-performing model and signal based on the highest `total_return`.
3.  Check the open trades endpoint to ensure no other tradeable instance of the same model and signal exists.
4.  **Crucially, it will execute a trade immediately if the open trade check passes, without performing any price comparison.**
5.  The `trade_reason` for trades generated by this script will be set to `"str_topPerformerCopy01_str_topPerformerAtAvgPrice"`.
