  imbalance threshold.
  - If it has no position and its cooldown has expired, it opens a new trade in the stable direction. The size is
  STANDARD_TRADE_SIZE (50000) clamped to the configured min/max lot sizes.

  Agent 02 sees either a BUY or SELL bias; otherwise the sentiment remains neutral and it won’t trade until a clear
  imbalance shows up (and is confirmed for the required number of ticks).
