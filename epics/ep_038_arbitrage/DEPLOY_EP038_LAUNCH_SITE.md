# EP038 Launch Site Deployment Notes

Static launch package: `site_launch/`.

## Launchable locally

Open `open_ep038_launch_site.bat` or `site_launch/index.html`.

## Public deployment options

1. Copy `site_launch/` to GitHub Pages.
2. Or upload `site_launch/` to Netlify/Vercel/static host.
3. Configure a real form/notification backend before collecting real subscriber details.

## Blockers before public launch

- User-owned hosting/domain decision.
- Operator/legal/privacy contact details.
- Notification provider for live alerts.
- Real spread data monitoring source decisions.
- Payment provider if charging.

No purchases, listings, external notifications, or paid actions are performed by this static package.
