# EP038 — Marketing / Launch Plan

Generated: 2026-06-28

## Product being launched

EP038 is a digital information product:

```text
Live pricing-spread intelligence for marketplace resellers and opportunity hunters.
```

The product publishes:

- total gross spread value identified;
- total count of spread candidates;
- amount of spread value expiring within a time window;
- live spread cards with price/source/risk/confidence/time pressure;
- subscriber preference alerts for selected categories and thresholds.

## Primary sales hook

```text
We found £X gross spread value across Y spread candidates — and £Z may expire within 24h.
```

Use “gross spread value”, not “profit”.

## Positioning statement

```text
EP038 monitors marketplace price gaps and turns them into a live information feed, so subscribers can see where spreads are appearing, how much value is visible, and which signals are time-sensitive.
```

## Audience segments

| Segment | Pain | Message |
|---|---|---|
| Side-hustle resellers | spend hours hunting manually | Stop hunting blind — watch live spread candidates in one board. |
| Trainer/fashion resellers | sizes and prices move quickly | See spread value and expiry pressure before listings disappear. |
| Domain investors | raw domain lists are noisy | Filtered domain spread candidates with time windows and risk notes. |
| Digital-product sellers | resale rights and policy are unclear | Spread candidates with rights/policy risk flagged, not hidden. |

## Launch wedge

Start broad but UK-focused:

```text
Live pricing-spread board for UK marketplace resellers.
```

Do not start with every marketplace/category. Use visible sample categories while asking users which feed they want first.

## Offer ladder

### Free preview

- delayed/sample spread board;
- methodology;
- preference capture;
- weekly public “spread value found” post.

### Founder access test

Suggested test price:

```text
£9–£19/month
```

Includes:

- active board;
- preference alerts;
- source/risk notes;
- category voting;
- early access to live monitoring.

### Pro later

- instant alerts;
- category feeds;
- history charts;
- private source links;
- expiry/auction notifications;
- niche boards.

## 14-day launch sequence

### Days 1–2 — Product proof

- Publish the static launch page.
- Show current sample board and clearly label sample/research state.
- Add waitlist/preference capture.
- Prepare 5 example spread cards/screenshots.

### Days 3–5 — Explain the concept

Content:

1. “What is gross spread value?”
2. “Why expiring spread value matters.”
3. “Why delisting does not always mean sold.”
4. “How EP038 avoids fake profit claims.”
5. “Choose your preferred spread alerts.”

CTA:

```text
Join early access / choose your categories.
```

### Days 6–10 — Organic discovery

Channels:

- X/Twitter reseller/side-hustle discussions;
- Reddit where allowed;
- Facebook groups only where rules allow and no spam;
- eBay/reseller communities;
- domain forums/NamePros for domain-specific angle;
- known contacts only if appropriate.

Ask questions before pitching:

```text
Would a board showing total spread value + expiry pressure be useful?
Which category would you want monitored first?
What minimum spread would make an alert worth seeing?
```

### Days 11–14 — Founder validation

Publish update:

```text
This week EP038 identified £X gross spread value across Y candidates; £Z was time-sensitive.
```

Offer founder access:

```text
First 20 users can request a preferred category feed.
```

## Constructive risks

- Sample data must not be mistaken for verified live spreads.
- Real monitoring source access may be blocked or require APIs/accounts.
- Alerts are valuable only if data freshness is credible.
- “Spread value” must never be framed as guaranteed profit.
- Public launch needs privacy/operator/payment details if collecting real users.
